# DecisionAI

**DecisionAI** is a smart AI-powered decision assistant that helps users weigh pros and cons and provides recommendations for personal or professional choices. Built with **Django**, **MongoDB**, and **OpenAI GPT API**.

## Features

- Enter a decision problem and get structured AI suggestions.
- Pros, Cons, Recommendation, and Confidence displayed clearly.
- User session management with login/logout.
- History tracking stored in MongoDB.
- Clean, responsive UI with Bootstrap 5.
